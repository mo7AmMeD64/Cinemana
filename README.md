# سينمانا 🎬

تطبيق Flutter لمشاهدة الأفلام والمسلسلات من Cinemana Shabakaty.

## المميزات
- مشغّل فيديو داخلي متكامل (media_kit)
- ترجمة SRT/VTT مع تخصيص كامل (حجم، لون، خلفية)
- اختيار جودة التشغيل
- تحكم بالسرعة (0.5x → 2x)
- double-tap للتقديم/الترجيع
- بحث، أقسام، مفضلة

## البناء
كل push على `main` يبني APK تلقائياً عبر GitHub Actions.
